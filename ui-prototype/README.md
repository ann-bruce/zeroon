# ZEROON Sprint 1 UI Prototype

Run:

```bash
python3 -m http.server 4173 --directory ui-prototype
```

Open `http://localhost:4173`.

Review scope:

- SMS login
- current-state home
- zero-record creation
- save confirmation
- private archive timeline
- companion growth metrics
- offline, AI failure, session expiry, and empty states

This prototype is for product and visual review. It does not call backend APIs.
