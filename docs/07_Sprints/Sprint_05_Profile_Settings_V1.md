# Sprint 05 Plan V1

Status: Draft

## Sprint Goal

Add a calm "My / Settings" foundation so ZEROON can understand the user's
self-provided context without becoming a social profile, diagnosis system, or
mandatory onboarding form.

The profile exists to help ZEROON respond with better continuity and relevance.
It is not a public identity page.

---

## Product Decision

ZEROON will add optional user profile settings as part of the V1.1 direction.

Confirmed scope:

- Add a My / Settings entry point.
- Let the user edit basic self-provided information.
- Let the user decide whether AI can use profile context.
- Keep all fields optional.
- Keep profile content private by default.

Do not add:

- Public profile pages
- Social display fields
- Required gender or exact age collection
- Personality labels generated by AI
- Diagnosis, scoring, or fixed user classification

---

## Profile Fields

| Field | Required | Notes |
|---|---|---|
| `nickname` | no | Used in local greeting and AI tone personalization |
| `avatar` | no | Sprint 05 can use default or preset avatars first |
| `age_range` | no | Prefer ranges over exact age |
| `occupation` | no | Free text, such as student, designer, founder |
| `self_description` | no | One short sentence about how the user wants ZEROON to understand them |
| `ai_profile_context_enabled` | yes | Explicit switch for whether AI can use profile context |

Avatar upload is recommended as a later enhancement unless storage, access
control, compression, and default fallback are all ready in this sprint.

---

## User Flow

1. User opens My / Settings.
2. User edits optional profile fields.
3. User sees a short explanation:
   - These details help ZEROON understand your records with more context.
   - You can leave them blank.
   - You can disable AI use of these details at any time.
4. User saves changes.
5. AI reflection uses profile context only when permission is enabled.

## Acceptance Criteria

- User can open My / Settings.
- User can edit and save optional profile fields.
- User can leave every profile field blank except the AI permission flag.
- ZEROON explains why profile context is collected.
- AI uses profile context only when the user enables it.
- Profile context improves wording and relevance but does not create fixed user labels.
- Profile data is private and owner-scoped.
- Avatar upload is not required for Sprint 05 completion.

---

## Out of Scope

- Full file upload and image processing pipeline
- Public identity pages
- Social sharing
- Gender as a required field
- Exact age as a required field
- AI-generated personality labels
- Long-term hidden psychological profile
